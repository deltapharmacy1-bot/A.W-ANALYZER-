# لوحة مقارنة أسعار الصيدلية (نسخة ويب مع قاعدة بيانات)

نسخة كاملة من لوحة تحكم التسعير والربحية، بعد إضافة سيرفر خفيف (Node.js)
وقاعدة بيانات SQLite لحفظ آخر بيانات مرفوعة تلقائياً، بحيث أي عضو في
الفريق يفتح الرابط من أي جهاز يلاقي نفس البيانات فوراً بدون ما يرفع
ملفات Excel من جديد.

## التشغيل محلياً

```bash
npm install
npm start
```

بعدها افتح: http://localhost:3000

## طريقة الحفظ

- التحليل والقراءة لملفات الإكسل لسه بتحصل في المتصفح بالظبط زي ما كانت
  (مفيش تغيير في منطق الحسابات).
- بعد أي رفع ناجح للملفات، البيانات الناتجة بتتبعت تلقائياً للسيرفر
  وتتخزن في `data/dashboard.db`.
- أول ما حد يفتح الصفحة، بيتم تحميل آخر نسخة محفوظة تلقائياً.
- زرار "إعادة رفع" بيسمح برفع ملفات جديدة تحل محل القديمة.

## النشر أونلاين (عشان الفريق يدخل من أي جهاز)

أبسط طريقة: نشر المشروع على منصة زي Render.com أو Railway.app (فيه باقات
مجانية/رخيصة):

1. ارفع المجلد ده على مستودع GitHub.
2. من Render/Railway: اعمل "New Web Service" واربطه بالمستودع.
3. Build command: `npm install` — Start command: `npm start`
4. لازم تضيف **Persistent Disk** (Render بيديها في الخطط المدفوعة، أو
   استخدم Railway اللي بيدي Volume) وتربطه بمسار `/data` جوه المشروع
   عشان قاعدة البيانات ميتمسحش مع كل نشر جديد.
5. بعد النشر هتاخد رابط زي `https://your-app.onrender.com` تقدر تبعته
   لأي حد في الفريق.

إذا حبيت، أقدر أساعدك تنشره فعلياً لو عندك حساب على إحدى المنصتين دول.
